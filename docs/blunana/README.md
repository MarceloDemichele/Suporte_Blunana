# Documentacao Blunana

Esta pasta deve concentrar documentacao funcional, tecnica e operacional relacionada ao Blunana.

Artefatos por tipo e ambiente devem ser gravados em:

- `outputs/logs/blunana/{ambiente}/`
- `outputs/json/blunana/{ambiente}/`
- `outputs/screenshots/blunana/{ambiente}/`
- `outputs/relatorios/blunana/{ambiente}/`
- `outputs/temp/blunana/{ambiente}/`

Nao misturar evidencias Blunana com evidencias dos robos.
