# Regras de negócio — Ateste

## Geração

Atualmente, o ateste é gerado **manualmente** pelos usuários para registrar atividades já executadas que devem ser cobradas da CEF.

Não existe regra automática em funcionamento para gerar atestes. Há previsão de um projeto futuro que poderá gerar atestes automaticamente quando determinados prazos forem cumpridos. Essa previsão não deve ser comunicada como funcionalidade disponível.

## Pagamento

A validação de pagamento utiliza o **Código de protocolo** informado no momento da solicitação do ateste à CEF.

O usuário também pode realizar upload do arquivo de pagamento para confirmar os pagamentos recebidos da CEF.

## Tipo de Ateste

Novos Tipos de Ateste devem ser criados exclusivamente na tela **Tipo de Ateste**.

## Histórico

O histórico dos atestes é mantido para fins de auditoria.

## Resposta recomendada sobre automação

> Não. Atualmente, os atestes são gerados manualmente pelos usuários. Não existe uma regra automática ativa para gerar atestes ao cumprir um prazo. Há apenas a previsão de um projeto futuro para automatizar a geração em determinados tipos de prazo.

## Fonte

Regras `RN-060` a `RN-065`, fornecidas e confirmadas pela equipe de suporte em 2026-07-15.
