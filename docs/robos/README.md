# Documentacao dos Robos

Esta pasta deve concentrar documentacao funcional, tecnica e operacional dos robos.

Artefatos por tipo e ambiente devem ser gravados em:

- `outputs/logs/robos/{ambiente}/`
- `outputs/json/robos/{ambiente}/`
- `outputs/screenshots/robos/{ambiente}/`
- `outputs/relatorios/robos/{ambiente}/`
- `outputs/temp/robos/{ambiente}/`

Nao misturar evidencias dos robos com evidencias do Blunana.
